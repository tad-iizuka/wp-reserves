== Description ==

Transform your WordPress installation into a nice Reservation System! Reserves manager can register items, configure and customize how reserves will work and manage reserves. Any user with login on the system can request an available reserve item on the system, and notified by e-mail during the proccess. This script is ideal for universities or small/medium companies to manage physical resources.

What an logged User can do:

* Request reserve item
* Manage, view or cancel their reserves
* View reserves history

What an Admin or Reserves manager can do:

* Manage reserves (authorize, reject, finalize...)
* Configure e-mail notifications
* Configure reserves page (filters, display...)
* Configure and view event logs

Features:

* Custom e-mail notifications
* Customizable reserves page
* Configurable log events
* Intuitive and easy to use

Demonstration URL: http://ragob.com/wpreserves (login: res/res)

**If you enjoy this plugin, and want future updates, please help clicking on Donate button.**

== Installation ==

1. Extract this plugin to your wordpress/wp-content/plugins directory
2. Go to the WordPress Admin Page, Plugins section
3. Enable the "Wordpress Reservervation System" plugin
4. A new menu will appear: WP Reserves
